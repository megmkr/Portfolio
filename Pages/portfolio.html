LinkedIn Data Scraping

I led the development of a Python-based web scraper to automate the collection of alumni employment data from LinkedIn, supplementing Point Loma Nazarene University’s traditional survey process. Using Selenium, I simulated browser behavior to locate and access alumni profiles, followed by BeautifulSoup for HTML parsing and Pandas for structuring the data. To navigate LinkedIn’s bot detection and HTML inconsistencies, I implemented conditional logic and timing strategies that stabilized scraping performance across nearly 1,000 profiles. I also conducted extensive debugging and testing to ensure accurate matching by city and education data, even when alumni didn’t list the university explicitly.

The project output was a clean, structured Excel dataset containing names, cities, job history, and profile metadata — ready for analysis and institutional use. My work directly enabled the university to improve alumni engagement, support federal reporting, and better showcase career outcomes to current students. I applied data validation logic to distinguish between guaranteed alumni matches and likely-but-unverified ones, improving the reliability of the dataset. Looking ahead, this codebase can be extended to track alumni employment trends, calculate unemployment rates, and analyze alignment between degrees and career paths.

Hindcasting Water Supply

I developed a full-stack hindcasting tool to analyze and visualize historical reservoir storage patterns across the U.S. The project involved designing a PostgreSQL database to manage historical reservoir data, building a Python-based pipeline for preprocessing and training machine learning models, and delivering an interactive Django dashboard. This dashboard allows users to explore predicted and observed reservoir storage values by region, making hydrological insights accessible to researchers, policymakers, and the public. The model supports better water resource planning by revealing trends and deviations in reservoir behavior over time.

What's She Doing Now?

As the final project for Course 2 of the Google Advanced Data Analytics Certificate, I am working on a Python-based analysis of user churn for Waze. This project focuses on cleaning and inspecting real user data to uncover behavioral patterns between retained and churned users. Using pandas and NumPy, I explored driving habits, device usage, and potential user personas—like long-haul drivers—to help inform strategies for retention. This hands-on lab deepens my skills in EDA, hypothesis development, and data storytelling.
